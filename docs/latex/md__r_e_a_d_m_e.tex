Just write flag in command line interface and provide parameter\+:

1) {\bfseries{strategy}} -\/ using for determining execution strategy, can take one of the following values\+: 1) {\itshape high\+\_\+prior} -\/ runs only test cases with priority that equal to High; 2) {\itshape random} -\/ pseudo random strategy, shuffle test cases and execute them; 3) {\itshape parallel} -\/ parallel test case search; 4) {\itshape everything\+\_\+now} -\/ print all tests cases in console for fully manual mode; 5) {\itshape usual} -\/ usually test case execution (do nothing with ts); 6) {\itshape choose$<$$>$} -\/ you can enter strat name, by default it will be all tests (if you do not write triangular brackets) or you may enter brackets and utility give test cases only with chosen priority, other will be ignored. 2) {\bfseries{devices}} -\/ provide text file with devices to test on, 3) {\bfseries{time\+\_\+record}} -\/ bool value, if set to true then each test case will count of execution, 4) {\bfseries{colored}} -\/ bool value, if set to true then utility output text will be colored in colors, 5) {\bfseries{comments}} -\/ bool value, if set to true then enables comments in test cases. 6) {\bfseries{interface}} -\/ bool value, if set to true starts graphical user interface\hypertarget{md__r_e_a_d_m_e_autotoc_md4}{}\doxysection{\texorpdfstring{Symbols meaning\+:}{Symbols meaning\+:}}\label{md__r_e_a_d_m_e_autotoc_md4}
